## ğŸš€ How to Run it?

From project's root file:

### Backend 
Remember to keep it running while web/mobile application are being used
```sh
$ cd server
$ yarn

# Run migrations
$ npx prisma migrate dev
$ yarn dev

# Run prisma studio on another console
$ npx prisma studio
```

### Web application:

```sh
$ cd web
$ yarn install
$ yarn dev
```

### Mobile:

```sh
$ cd mobile
$ yarn install
$ npx expo start
```
